熟悉pycharm

pycharm的使用

创建一个项目

  1. 点击File -> New Project -> 设置保存位置 -> create
  2. 在project下创建项目文件，New -> Python File -> 输入名字 -> 回车
  3. 在程序页面输入
  4. 右键程序页面或者项目文件 Run ‘hello world’ 或者 ctrl+shift+F10 快捷键

其他常用快捷键

  • ctrl+d 快速复制当前行代码
  • ctrl+shift+上下键 上下移动选中代码
  • shift+enter 快速换行（光标在代码中也可以切换到新的一行，不用挪到代码最后）
